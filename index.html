DonggriArchive Cloudflare Blog Template

실제 DonggriArchive 워크스페이스 구조를 기준으로 정리한 공유용 템플릿입니다. 개인정보/운영키/리소스 식별자는 제거하고, 바로 GitHub Template과 GitHub Pages 소개 페이지로 사용할 수 있게 구성했습니다.

Repository README

템플릿 구조

  • apps/web: 공개 블로그 프론트엔드
  • apps/admin: 관리자 콘솔
  • apps/api: Cloudflare Worker API
  • packages/shared: 공통 타입/유틸
  • docs: 운영/구조 문서
  • scripts: 시드/검증 스크립트
공개 가능
템플릿 코드
문서/예시값
비공개 필수
실 도메인
Cloudflare IDs
API Token / 실제 .env

핵심 기능

  • Public posts / categories / tags / search
  • Admin login/session + 콘텐츠 CRUD
  • RSS / sitemap XML / asset proxy
  • Cloudflare Pages + Workers 연동 구조
  • 모노레포(pnpm workspace) 기반
  • 로컬 검증 스크립트 포함
  • GitHub Template 공유에 맞춘 기본값 구성
  • 운영값 분리(private repo) 전략 반영
  • 자동화·SEO·CTR 중심 운영에 맞춘 API 설계

추천 워크플로우

  • 1) public 템플릿 저장소에서 공유
  • 2) Use this template로 새 private repo 생성
  • 3) private repo에서 운영값만 주입
  • 4) Cloudflare Pages/Workers는 private repo 연결

WSL 실행 예시

cd /mnt/d/Donggri_Platform/cloudflare-blog
pnpm install
pwsh ./scripts/setup-local-dev.ps1
pnpm dev:api
pnpm dev:web
pnpm dev:admin
